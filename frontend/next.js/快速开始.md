### 全局安装create-next-app工具

1. 通过npm安装

```bash
npm install create-next-app -g
create-next-app nextproject  //创建next项目
```

2. 通过npx直接创建项目

```bash
npx create-next-app --use-npm --example https://github.com/vercel/next-learn-starter/tree/master/learn-starter
```
