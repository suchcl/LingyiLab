### 高频使用的数据处理方式

#### 数据过滤

```ts
const data = [
    {
        name: 'zhangsan',
        age: 22,
        address: {
            city: 'New York',
            country: 'USA'
        }
    },
    {
        name: 'lisi',
        age: 42,
        address: {
            city: 'Paris',
            country: 'France'
        }
    },
    {
        name: 'wangwu',
        age: 17,
        address: {
            city: 'London',
            country: 'UK'
        }
    }
];
const lastData = data.filter(item => item.address.country === "USA")
console.log('%c [ lastData ]-39', 'font-size:13px; background:pink; color:#bf2c9f;', lastData)
```

filter是数组中的一个标准方法，会将符合条件的数据返回，案例中的返回数据为：

```ts
[
    {
        name: 'zhangsan',
        age: 22,
        address: {
            city: 'New York',
            country: 'USA'
        }
    }
]
```
