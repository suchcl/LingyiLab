### js中的函数注释

js中函数常用的注释风格，有JSDoc和Google风格两种类型，也可能还有其他类型，但是从个人使用习惯比较倾向于JSDoc这种风格，格式如下：

```js
/**
 * [函数/变量描述]
 *
 * @param {参数类型} 参数名 - 参数描述
 * @returns {返回值类型} 返回值描述
 * @throws {异常类型} 异常描述
 */
```

案例如下：

```js
/**
 * 计算两个数字的和
 *
 * @param {number} a - 第一个加数
 * @param {number} b - 第二个加数
 * @returns {number} 两个数字的和
 */
function add(a, b) {
  return a + b;
}
```
