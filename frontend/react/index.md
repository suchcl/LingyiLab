### React

React[https://zh-hans.reactjs.org/](https://zh-hans.reactjs.org/),是一个facebook退出的开源UI库，聚焦在UI层，在前端构建用户界面，React和传统前端技术的优势是可以创建组件，这个组件是可自定义、可复用的HTML元素，可以更快、更高效的构建用户界面。另外，React还可以使用state和props简化数据的存储和处理方式。
