#### 1.1 pnpm的安装

pnpm有多种安装方式，常用的可以通过前端的包管理工具如npm、yarn来安装，也可以通过独立脚本安装。

**通过包管理工具来装**

1. 通过npm安装

```bash
npm install pnpm -g
```

或者

```bash
npm install @pnpm/exe -g
```

2. 通过yarn安装

```bash
yarn add pnpm
```
