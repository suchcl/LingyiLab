1. 合并两个有序数组

给你两个按非递减排序排列的整数数组nums1和nums2,另有两个整数m和n,分别表示nums1和nums2中元素的数目。

请你合并nums2到nums1中,使合并后的数组同样按照非递减顺序排列。

```js
function merge(arr1, m, arr2, n) {
    let i = m - 1;
    let j = n - 1;
    let k = m + n -1;
    while(j >= 0){
        if(i>=0 && arr1[i] > arr2[j]){
            arr1[k--] = arr1[i--];
        }else {
            arr1[k--] = arr2[j--];
        }
    }
}
```
